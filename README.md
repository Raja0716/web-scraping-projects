# 🛍️ Amazon Product Scraper

This Python project scrapes product information from an Amazon product page using `BeautifulSoup`, `requests`, and `lxml`. The data is exported to a CSV file for further analysis or reporting.

## 📌 Features

- Extracts product:
  - Title
  - Price
  - Rating
  - Bullet points (key features)
  - Description
- Saves data to a CSV file (`amazon_product.csv`)
- Uses Python standard libraries

---

## 🛠️ Tech Stack

- Python
- BeautifulSoup4
- Requests
- lxml
- CSV module
